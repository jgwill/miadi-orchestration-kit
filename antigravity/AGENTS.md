# Antigravity Agent & Plugin Specifications

This directory holds the agent specifications, prompt contracts, and operational scripts for **Google DeepMind Antigravity** within the Miadi Orchestration Kit framework.

Unlike standard static prompt companions or CLI-based wrappers, Antigravity companions leverage the assistant's fully agentic toolset to interactively drive workflows, automate validation, and generate visual narrative pre-production assets.

## Capabilities

1. **Agentic Execution**: Direct file reading, folder initialization, and automated workspace routing.
2. **Visual pre-production**: Live illustration and creation of visual chronicle seeds using native `generate_image` integration.
3. **Proactive Verification**: Automated execution of test suites and validation scripts via local shell runtimes.

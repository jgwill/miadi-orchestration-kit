Structural Tension View

per rispec layer
📋 Agent Roster
Current Reality
Story work blurs drafting, critique, editing, continuity, and protocol judgment into one undifferentiated assistant voice.
Desired Outcome
A clear writing team whose members each carry a distinct narrative responsibility.
Natural Progression
Define agents by mandate, inputs, outputs, and handoff boundaries so the Copilot kit can coordinate specialized story work.
🎯 Skill Surface
Current Reality
Agent roles alone do not create repeatable workflows; users must remember every agent and artefact path.
Desired Outcome
A compact set of Copilot skills that can run the storytelling pipeline without requiring coordination knowledge from the user.
Natural Progression
Define each skill by trigger, workflow, required inputs, artefact outputs, and agent collaboration.
📊 Pipeline
Current Reality
No reliable order of operations: brief, drafting, review, and export happen ad-hoc with no tracked state transitions.
Desired Outcome
A RISE-aligned pipeline where each phase produces artefacts that gate the next, with clear resume points.
Natural Progression
Brief → Bible → Outline → Chapters → Review → Export, with Continuity Keeper and Protocol Steward as cross-cutting threads.
🔄 State & Handoff
Current Reality
Each new prompt restarts context. Agents cannot resume from a prior session's artefacts without re-reading everything.
Desired Outcome
A state manifest (state.md) that any agent can read to know what's been accepted, what's pending, and what's blocked.
Natural Progression
Orchestration Architect owns state.md. Every skill that touches artefacts also updates state. Review gates write their own route recommendations.
🛡️ Cultural Protocol
Current Reality
Sensitive material (Indigenous knowledge, sacred content, personal trauma) flows through the pipeline without explicit pause or authority review.
Desired Outcome
A protocol steward that can pause any skill, names what it does not know, and routes to human guidance before proceeding.
Natural Progression
Define steward authority, consent boundary types, and four-route output (continue / with-boundaries / ask-human / do-not-proceed).
📖 Episode Memory
Current Reality
Sessions produce drafts but lose story-context: characters, world facts, and narrative promises evaporate when the window closes.
Desired Outcome
Each session produces an episode packet (StoryForms, StoryBeats, Settings) that feeds back into Miadi's world memory for future sessions.
Natural Progression
Export Steward assembles packet. Continuity Keeper validates it. Closure note records follow-up commissions and open story promises.
